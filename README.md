## 開發請往 `dev` 分支

請在 Repository 被 Clone 下來之後進行分支切換。

由於 `main` 和 `dev` 預設只有 Pull Request 才可以合併，故請在切換分支後馬上開一個新分支進行開發，並遵守 [Git FLow](docs/GitFlow.md)

```shell
git switch dev
git branch feature/XXX
git switch feature/XXX
```
